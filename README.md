## Getting Started

### First, install dependencies:

```bash
npm install
# or
yarn install
```

### Run the development server:

```bash
npm run dev
# or
yarn dev
```

### Run production server:

```bash
# first
npm run build
# or
yarn build
# second
npm start
# or
yarn start
```
